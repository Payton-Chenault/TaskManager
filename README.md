# Java Task Manager

Java Task Manager (JTM) was a learning project in order to analyze and understand CRUD operations and how to make Java and SQLLite interact togeather.

## Installation

To use the app, download the .exe found in the releases, other O.S. has not been compliled to support right off the download.
